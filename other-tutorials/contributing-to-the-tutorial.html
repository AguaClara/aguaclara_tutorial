Contributing to the Tutorial¶

  1. Go to the aguaclara_tutorial repository’s issues page.

  2. Open a “New Issue”, detailing what you would like to add (or what you would like to see added) to the tutorial.

  3. Add the label “addition” and tag oliver-leung to the issue body.
